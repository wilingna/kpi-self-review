# KPI 绩效自评 Skill

> 把零散的工作成果，变成让 HR 和老板眼前一亮的绩效语言
https://wilingna.github.io/kpi-self-review/
适用于 Claude / Open Claude

---

## 效果演示

| 步骤 | 你做的事 | AI帮你做的事 |
|------|----------|--------------|
| 上传KPI表 | 截图/拍照上传 | 自动识别所有指标、子指标、权重 |
| 确认结构 | 检查一眼，按需微调 | — |
| 填完成情况 | 写几个关键词和数据 | 生成正式的绩效语言描述 |
| 打分 | 拖滑块调整 | 自动加权计算总分 |
| 导出 | 点一下 | 生成带格式的Excel表 |

**从上传到导出，全程约5-10分钟**（手写自评通常需要1-2小时）

---

## 适用场景

- ✅ 定性指标为主（"推进XX落地"、"提升XX满意度"）
- ✅ 定量指标为主（目标值 vs 实际值）
- ✅ 定性+定量混合（最常见的职场情况）
- ✅ 多层级子指标（大类下有具体子条目）
- ✅ 自定义权重（实时校验总和是否=100%）
- ✅ 百分制打分（60-100分，加权算总分）
- ✅ 任何岗位：HR、产品、技术、运营、市场、行政……

## 使用方式

### 方式一：直接用 Claude（最低门槛）

把 `SKILL.md` 复制到 Claude 的 Project 说明里，直接对话即可。

### 方式二：本地打开 HTML 工具

下载 `kpi-pro-universal.html`，Chrome 直接打开，填入 API Key 即可使用。

### 方式三：部署到线上

参考 `DEPLOY.md`，GitHub Pages + Cloudflare Worker，约15分钟完成。
